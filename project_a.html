Citymapper

UX, UI, and Interaction Design

Project Context

For this project, we worked with an existing mobile application, Citymapper, and designed a new feature for it. Citymapper is transit and mapping app that helps users navigate around the city and guide them on transportation. Citymapper can be used on multiple platforms and is available for different major cities around the world.

Project Type

Academic project - case study

Role

UI design, UX design, interaction design, visual design, branding + identity

Tools

Adobe Illustrator, Photoshop, XD
